## Technologies

1. Vue3 + Typescript
2. Tailwind 
3. [Figma](https://www.figma.com/file/FeKRiWESyfm0k4JuSvGoP9/Tip-Calculator?node-id=0%3A1)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
